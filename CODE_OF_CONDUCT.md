# Code of Conduct

This project follows the canonical Tessera Code of Conduct published in
[`tessera-llm/tessera-sdk`](https://github.com/tessera-llm/tessera-sdk/blob/main/CODE_OF_CONDUCT.md).

In short: be respectful, assume good faith, no harassment, no discrimination,
no off-topic noise. Maintainers may remove comments, commits, or contributors
that violate the spirit of those rules.

**Enforcement contact:** [conduct@tesseraai.io](mailto:conduct@tesseraai.io)

Tessera is operated by Fintechagency OÜ, Tallinn, Estonia (registry 16638667).
